# Customer Churn Prediction & Customer Lifetime Value (LTV) Engine

## Overview

An end-to-end Data Analytics and Machine Learning project that predicts customer churn and estimates Customer Lifetime Value (LTV) using SQL, Python, PostgreSQL, XGBoost, FastAPI, and Power BI.
